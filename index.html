Borno CMS

This is the free version of Borno CMS

Require

  • PHP/MYSQL
  • Apache
  • Apache rewrite mod must be enable

Progress

  1. Download the source code and upload it in your directory
  2. Create a Database
  3. Run http://youdirectory/install.php
  4. Fill up the form
  5. Click Submit

Login and ready your site from http://yourdirectroy/admin/
